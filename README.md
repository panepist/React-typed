# Typed - React JS
-----

Typed is a component written in JavaScript. This component makes text effects and specifically displays text like typing someone.

# Hints on Usage
-----
```
import {Typed} from './typed';
export function App(){
    ...
    return (
        ...
        <Typed msg="...text..."/>
        ...
    )}
```
